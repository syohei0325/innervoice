# PARTNER MODE – 同意ベース共有

## 目的
パートナー（恋人/配偶者/家族）との予定調整を**最小共有**で実現。

## 原則
- **同意が核**：共有は最小・期限付き・撤回自由
- **必要最小**：NG食材/価格帯/雰囲気/移動手段/当日位置のみ
- **透明性**：誰が何を共有/利用したかを90日保持（削除/エクスポート可）

## 共有例
### 飲食店選び
- **NG食材**：アレルギー/苦手
- **価格帯**：予算上限
- **雰囲気**：静か/賑やか
- **移動手段**：徒歩/車/公共交通

### 当日調整
- **位置**：現在地（必要時のみ・期限付き）
- **空き時間**：カレンダーの空きスロット

## 実装
### Consent管理
- `POST /api/consent` → 同意を記録
- `GET /api/consent` → 同意一覧を取得
- `DELETE /api/consent/:id` → 同意を撤回

### データ構造
```json
{
  "id": "consent_123",
  "user_id": "user_1",
  "partner_id": "user_2",
  "scopes": ["ng_food", "price_range", "location"],
  "expires_at": "2025-12-31T23:59:59Z",
  "created_at": "2025-10-20T10:00:00Z",
  "revoked_at": null
}
```

## 監査
- **誰が何を共有したか**：90日保持
- **誰が何を利用したか**：90日保持
- **削除/エクスポート**：いつでも可能

## KPI
- Partner利用率
- 共有範囲の最小性（スコープ数の中央値）
- 撤回率（低いほど良い）

## プライバシー
- 本文は共有しない（要約＋メタのみ）
- 期限付き共有（既定30日）
- 撤回自由（即時反映）
